##古卡老屋

基于thinkphp3.2框架开发
由于一些内部原因，古卡管理系统要重新开发，去掉了点餐的功能，以及和点餐相对应的菜单管理和后台的订单打印等相关功能都去掉了。
现在主要开发手机版的web，后期开发使用跨平台语言apicloud进行移动端的开发。

古卡老屋管理网站，主要用于古卡老屋的日常经营管理。
主要有四大模块
1、兼职模块，细分为兼职签到和兼职的日常工作的管理。
2、原料采购，主要提供原理的日常采购，原理的入库。
3、财务模块，日常财务的记录的分析总结。
